# Product Builder Lecture

An interactive, kid-friendly learning platform featuring a cute mascot and a top-down learning map.

## 🚀 Live Demo
You can view the live site here: [https://tommyt200.github.io/product-builder-lec/](https://tommyt200.github.io/product-builder-lec/)

## Features
- **Vibrant & Bubbly UI**: Designed specifically for children.
- **Interactive Mascot**: Guides users through their learning journey.
- **Top-Down Map**: Visualizes the curriculum as an interactive adventure.
- **Automatic Deployment**: Powered by GitHub Actions.
